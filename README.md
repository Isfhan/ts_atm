# Typescript ATM

This ATM is an command line interface app that is using [TypeScipt](https://www.typescriptlang.org/), [Node.js](https://nodejs.org/en/) and [Inquirer](https://www.npmjs.com/package/inquirer) and [Chalk](https://www.npmjs.com/package/chalk) project that allows the user to guess the number generated by the computer.
